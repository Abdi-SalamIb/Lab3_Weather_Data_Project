# Lab 3 - VäderData Projekt

**Student:** Abdi-Salam Ibrahim  
**Kurs:** DevSecOps KYH 2025

## Beskrivning

Väderdata analys med C# och SQLite (Oktober-November 2016)

## Funktioner


- CSV import till databas
- Temperatur och luftfuktighetsanalys
- Mögelriskberäkning
- Meteorologiska säsonger
- Balkondörr-detektering
- Varaktighetsberäkning
- Temperaturjämförelser inne/ute

## Teknologier

- C# (.NET 8)
- SQLite
- LINQ

## Resultat

- 156,534 dataposter
- 58 dagar analyserade
- 1 balkondörröppning detekterad
- Mögelriskformel: `Risk = (0.5 × T) + (0.8 × RH)`

## Användning

1. Öppna i Visual Studio
2. Kör (F5)
3. Resultat visas i konsolen
